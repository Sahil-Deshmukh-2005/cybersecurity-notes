# Web Fundamentals

***

## HTTP 
`HTTP` or `HyperText Transfer Protocol` is used as a communication protocol to transfer data across the internet.  

***

## HTTPS 
`HTTPS` or `HTTP Secure` is a secure version of `HTTP`. Its used to encrypt the data while transfer or recieve.  

***

## HTTP Methods
These methods are used to state the users intent for the resources.  
1) GET = Retrive a Resource/Data
2) POST = Create(send data to server)
3) PUT = Update Resource/Existing data
4) PATCH = Partially Update Resource
5) DELETE = Deletes the Resource

***

## HTTP Requests
How the requests look like when an user wants to asscesses the resource.  
1) Version Type
2) URL
3) Method
4) Requests Header -> Extra information for servers
5) Body (optional) -> Input data

***

## HTTP Responses
How the response looks like when thw server responses to the request made by user.  
1) Requested Resources
2) Content Length
3) Content Type
4) Headers
5) ETags
6) Time Last Modified
7) HTTP Status Codes

***

## Status Codes 
These codes states the type of the connection between user ans server.  
```
100 - 199 ➡️ Informational
200 - 299 ➡️ Successful
300 - 399 ➡️ Redirection
400 - 499 ➡️ Client error
500 - 599 ➡️ Server error
```
