# shutil Module

shutil = Shell Utilities

I used it to move the file from one directory to other.  

## Working 
- Im sorting/organizing the files on the basis of their suffix.  
  ex., `.txt`, `.pdf`, `.docx`, `.png`, `.py`, etc ..  
  
- Im keeping the log gile i.e. `.log` to keep the track of the files being moved.  
  We can also read the log, or delete it before doing next organizing.  
  
- We are also handling the duplicate file name and file type.  
  
- We are also handling error that comes when moving root files, currently open files, ...  
  
- We are returning the statistical data for every moved, error and duplicate file.
