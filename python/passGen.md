## String Module/Library
String Library has built in functions/methods.  
ex.,  
```
    string.ascii_lowercase = returns string of lower case letters
    string.ascii_uppercase = returns string of upper case letters
    string.digits = returns string of numeric digits 0-9
    string.punctuation = returns string of symbols
```
