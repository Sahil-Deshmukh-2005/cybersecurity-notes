# Port Scanner

## Socket
Socket is pythons built in library which deals with:  
- Endpoints: A socket is essentially an endpoint combination of an IP address and a port number.  
- IP Address: Identifies a specific machine on a network.  
- Port Number: Identifies a specific application or process running on that machine.  

## Errno
Errno is a python built in library which translates system error code into human readable format.
