# Module-1 : Introduction to Cybersecurity

## Room-3 : Search Skills

***

Websites and Services that can be used to gather information for a variety of cyber security purpose, both offensive and defensive.  

***

1) Shodan :

`Shodan` is often described as search engine for Internet Of Things (IOT), but thats not it.  
`Shodan` continously scans the internet, searching for networking equipment, industrial control systems, traffic cameras and virtually anything else with public network connection to see whats running and where.  

***

2) VirusTotal :

`VirusTotal` collects results from over 70 antivirus engines and website scanners into a single interface.  
Submit a file, an URL, a domain or a file hash. VirusTotal will tell you whether any of these engines have flagged it as malicious or not.  

***

3) Vulnerability Databases (CVE) :

The `CVE` or `Common Vulnerability and Exposure` is like a universal directory for known vulnerabilities.  
Each confirmed vulnerabilities are assigned with a unique name format, `CVE-Year-Number`, ex., `CVE-2026-55197`.  
Each vulnerabilty is assigned with the `cvss` score. `CVSS = Common Vulnerability Scoring System`

***

4) Technical Documentation :

Meaning Official Documentation or Manual for the commands.  
```bash
man <command>
```
This prints the manual or the command on the terminal itself.  

***

5) Github :

And ofcourse `Github`, it is a great resourse to stay updated with the latest threats and vulnerabilities.  
`Github` also provides proof-of-concept (PoC) code, exploitation tool, and detailed technical report.
