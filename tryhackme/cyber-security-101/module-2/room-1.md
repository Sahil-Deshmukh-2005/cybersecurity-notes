Operator/Symbols and their Description:  
`&` = This is used to run commands in background.  
`&&` = This is used to rum multiple commands in one line into your terminal.  
`>` = This is the `redirector` meaning we can use to take the output of one command and direct it elsewhere.  
`>>` = This does same as `>` but instead of overwritting the file it appends the output to previous ones.
